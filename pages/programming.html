Project: Hard Bullet Mods

Hard Bullet's modding community, while small, has made a great number of mods which add content to the game, or adjust already existing content. All of the mods I have made can be found from my profile, beware the game is.. quite gory.


You can find all of the other mods created by people on the Thunderstore page, I would recommend taking a look at it. Just know that none of us are graphic designers.



Project: Termoculus

Oculus has always required its software to be open, within reason, when your VR headset is on. But whenever you turn it off, the software is still running, and constantly trying to re-launch itself when closed via task manager. There are other software solutions for this, but I was too lazy to deal with downloading those.


Instead, I made a program called 'Termoculus' which simply closes all 4 parts of the program at the same time. However, this is a personal program which I don't intent on releasing.
