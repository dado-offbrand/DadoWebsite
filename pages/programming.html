Project: Termoculus

Oculus has always required its software to be open, within reason, when your VR headset is on. But whenever you turn it off, the software is still running, and constantly trying to re-launch itself when closed via task manager. There are other software solutions for this, but I was too lazy to deal with downloading those.


Instead, I made a program called 'Termoculus' which simply closes all 4 parts of the program at the same time. However, this is a personal program which I don't intent on releasing.



Project: qStart

qStart was just a quick program used for starting certain applications in certain scenarios. For instance, Spotify, Visual Studio, and Chrome for coding. It lets you add paths to a text file from the application, and then lets you save/close the file. From there, just press start and it will start everything for you. Quicker than launching all 3, for the most part.


Currently there is no download for the application as I dropped it for a long while until recently, and I want to redo the whole application.
